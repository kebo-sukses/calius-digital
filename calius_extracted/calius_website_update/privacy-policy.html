4. Validasi Lisensi

Untuk validasi lisensi, kami menggunakan sistem berikut:

4.1 Hardware ID (HWID)

HWID dihasilkan dari kombinasi:

  • Processor ID
  • Motherboard Serial Number
  • MAC Address

Data ini di-hash menggunakan SHA-256 dan hanya disimpan secara lokal. HWID digunakan untuk memastikan lisensi tidak disalahgunakan.

4.2 Koneksi Server (Opsional)

Aplikasi dapat terhubung ke server kami untuk:

  • Validasi lisensi saat aktivasi
  • Pengecekan pembaruan (auto-update)

Koneksi ini tidak mengirimkan data personal atau konten video Anda.

5. Layanan Pihak Ketiga

Jika Anda mengaktifkan integrasi dengan layanan pihak ketiga, data mungkin diproses oleh mereka:

5.1 Google Gemini API

  • Data yang dikirim: Metadata video (judul, deskripsi) untuk analisis AI
  • Tujuan: Mengidentifikasi momen viral untuk clips
  • Kebijakan Google: Google Privacy Policy

5.2 Groq API (Whisper Transcription)

  • Data yang dikirim: File audio untuk transkripsi
  • Tujuan: Menghasilkan subtitle
  • Kebijakan Groq: Groq Privacy Policy

6. Keamanan Data

Kami menerapkan langkah-langkah keamanan berikut:

  • Enkripsi file lisensi: License file dienkripsi dengan XOR + HMAC untuk mencegah tampering
  • No plain-text storage: API keys dan data sensitif tidak disimpan dalam plain text di production build
  • HTTPS-only: Semua koneksi ke server kami menggunakan HTTPS
  • No logging: Kami tidak menyimpan log aktivitas pengguna

7. Privasi Anak-Anak

ClipPremium tidak ditujukan untuk anak-anak di bawah 13 tahun. Kami tidak secara sengaja mengumpulkan informasi dari anak-anak. Jika Anda orang tua atau wali dan percaya bahwa anak Anda telah memberikan informasi kepada kami, silakan hubungi kami.

  • Hapus: Menghapus semua data ClipPremium dari komputer Anda
