Troubleshooting

File download langsung terhapus/diblokir

Jika file langsung terhapus setelah download:

  • Matikan sementara antivirus (Windows Defender/third-party)
  • Tambahkan folder Downloads ke whitelist antivirus
  • Coba download ulang file
  • Restore file dari karantina antivirus jika diperlukan
Tidak ada opsi "Keep" atau "Keep anyway"

Jika opsi Keep tidak muncul:

  • Pastikan Anda mengklik tanda panah (^) atau titik tiga (...) di samping notifikasi
  • Coba klik kanan pada notifikasi download
  • Buka halaman Downloads browser (Ctrl+J) dan cari opsi di sana
  • Coba gunakan browser lain sebagai alternatif
Download sangat lambat atau gagal

Untuk masalah kecepatan download:

  • Periksa koneksi internet Anda
  • Matikan VPN jika sedang digunakan
  • Coba di waktu yang berbeda (hindari jam sibuk)
  • Clear cache browser dan coba lagi
  • Gunakan download manager seperti IDM atau Free Download Manager
"Windows protected your PC" saat instalasi

Ini adalah Windows SmartScreen protection:

  • Klik "More info" pada dialog yang muncul
  • Tombol "Run anyway" akan muncul
  • Klik "Run anyway" untuk melanjutkan instalasi
  • Ini normal untuk aplikasi baru yang belum memiliki sertifikat digital populer
Antivirus memblokir atau menghapus file

Untuk mengatasi false positive antivirus:

  • Windows Defender: Settings ‚Üí Virus & threat protection ‚Üí Exclusions ‚Üí Add ClipPremium folder
  • Avast/AVG: Settings ‚Üí Exceptions ‚Üí Add file path
  • Kaspersky: Settings ‚Üí Additional ‚Üí Threats and Exclusions ‚Üí Add
  • Norton: Settings ‚Üí Antivirus ‚Üí Scans and Risks ‚Üí Add exception
  • Atau matikan sementara real-time protection selama instalasi
