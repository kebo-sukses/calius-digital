## 🔧 TECHNICAL SEO CHECKLIST

### On-Page SEO
```
□ Title tag optimization (60 karakter, keyword di awal)
□ Meta description compelling (155 karakter, include CTA)
□ H1 tag = Main keyword (hanya 1x per page)
□ H2-H6 structured hierarchy
□ URL slug pendek & descriptive
□ Alt text semua gambar (include keyword naturally)
□ Internal linking min 5 per artikel
□ External links ke authority sites (2-3)
□ Image optimization (<100KB, WebP format)
□ Mobile responsive
□ Page speed <2s (Core Web Vitals)
```

### Schema Markup (Structured Data)
```json
{
  "@context": "https://schema.org",
  "@type": "Article",
  "headline": "Panduan Lengkap Landing Page 2026",
  "author": {
    "@type": "Person",
    "name": "Jeslius",
    "url": "https://www.calius.digital/about"
  },
  "publisher": {
    "@type": "Organization",
    "name": "Calius Digital",
    "logo": {
      "@type": "ImageObject",
      "url": "https://www.calius.digital/logo.png"
    }
  },
  "datePublished": "2026-05-27",
  "dateModified": "2026-05-27",
  "image": "https://www.calius.digital/images/landing-page-guide.jpg"
}
```

**Schema Types to Implement:**
- Article schema (semua blog post)
- FAQ schema (section FAQ)
- HowTo schema (tutorial articles)
- Product schema (template pages)
- Review schema (case studies)
- Organization schema (homepage)

### Core Web Vitals Optimization
```
Target Metrics:
- LCP (Largest Contentful Paint): < 2.5s
- FID (First Input Delay): < 100ms
- CLS (Cumulative Layout Shift): < 0.1

Actions:
1. Lazy load images
2. Defer JavaScript
3. Minify CSS/JS
4. Use CDN (Vercel Edge Network)
5. Optimize fonts (preload, font-display: swap)
6. Reduce server response time
```
